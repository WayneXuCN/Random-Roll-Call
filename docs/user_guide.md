# 随机点名软件使用说明

## 1. 软件简介
随机点名软件是一款专为课堂教学设计的工具，支持从Excel导入学生名单，随机抽取学生姓名，并记录点名历史。

## 2. 功能特性
- Excel导入：支持.xlsx和.xls格式文件导入学生名单
- 随机点名：支持1-20人同时点名
- 防重复模式：可选择是否允许同轮次重复抽取
- 动画效果：提供平滑的随机滚动动画
- 历史记录：记录所有点名结果
- 数据统计：提供点名次数统计
- 重名处理：智能检测和处理重复姓名
- 手动管理：支持手动添加和移除学生姓名

## 3. Excel模板格式要求
- 文件格式：.xlsx 或 .xls
- 表头：第一列应为"姓名"或其他标识学生姓名的列名
- 数据：每行一个学生姓名，支持最多1000个学生
- 编码：建议使用UTF-8编码

### 模板示例：
| 姓名   |
|--------|
| 张三   |
| 李四   |
| 王五   |
| ...    |

## 4. 快速开始
1. 启动软件：`uv run python -m src.main`
2. 点击"导入名单"按钮，选择Excel文件
3. 设置点名人数（1-20人）
4. 选择是否启用"防重复点名"
5. 点击"开始点名"按钮
6. 点击"停止点名"获取结果

## 5. 按钮功能说明
- **导入名单**：从Excel文件导入学生姓名
- **手动添加姓名**：手动输入学生姓名，支持一次输入多个，用逗号分隔
- **移除选中姓名**：从学生列表中移除选中的姓名
- **开始点名**：开始随机点名动画
- **停止点名**：停止动画并显示点名结果
- **重置名单**：清空当前学生名单
- **查看详细历史**：查看所有点名记录
- **清空历史**：清空所有点名记录

## 6. 菜单功能
- **文件**：清空历史记录
- **工具**：查看统计信息、清空学生名单

## 7. 数据存储
- 学生名单：保存在 `data/students.json`
- 历史记录：保存在 `data/history.json`
- 配置信息：保存在 `data/config.json`

## 7. 常见问题
Q: 为什么导入Excel失败？
A: 请检查文件格式是否为.xlsx或.xls，且第一列包含学生姓名。

Q: 如何确保每个学生被选中概率均等？
A: 软件使用Python内置的random模块，保证随机性。

Q: 软件支持多少学生？
A: 单个班级最多支持1000名学生。
